Git token:
--------
Go to GitHub Settings:
Visit: https://github.com/settings/tokens
Or: GitHub → Profile Picture → Settings → Developer settings → Personal access tokens → Tokens (classic)
Generate New Token:
Click "Generate new token (classic)"
Give it a descriptive name like "ReflexAI PR Analyzer"
Set expiration (recommend 30-90 days for hackathon)
Select Scopes/Permissions:
✅ repo (Full control of private repositories)
✅ public_repo (Access public repositories) 
✅ read:org (Read org and team membership)


OpenAI API Key:
-------

Visit: https://platform.openai.com/api-keys
Create API Key:
Click "Create new secret key"
Give it a name like "ReflexAI Hackathon"
Copy the key immediately
